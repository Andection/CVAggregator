Основные функции программы:
1. Загрузка данных с сайта e1.
2. Отображение основной информации из загруженных резюме.
3. Поиск.
4. Возможность перехода к полному резюме на сайт e1.

В качестве базы данных используется mongodb. Для запуска выполните startmongo.bat из директории mongodb. Внешние зависимости в solution устанавливаются через nuget
